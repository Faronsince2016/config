The Box

Box model

在 CSS 中， Box model 用于描述构成网页的元素，即一个个正方形 box.

本质上，一切皆为正方形。

一些有趣的效果:

  • border-radius 使 box 的四个角更平滑
  • box-shadow 添加阴影 (x偏移，y偏移，模糊度，颜色)
  • outline 和 box-shadow 并不是 box 的一部分，所以不会影响布局。

Box sizing

box-sizing 属性用于控制 box 的大小。两个值可选：

  • content-box: 默认值。计算 box 的宽高时，padding, border 需要额外加上。
  • border-box: padding, border 已包含在 box 的宽高内。注意： margin 不在这之内。最适宜场景就是把一个区域按百分比划分为3列。

Example:

height: 100px; width: 100px; padding: 20px; border: 5px solid black; margin: 5px;

推荐在所有 CSS 之前加入：(Bootstrap 3 已采用) 

      *, *:before, *:after {
        -webkit-box-sizing: border-box; /* Needed for mobile WebKit */
        -moz-box-sizing: border-box; /* Needed for Firefox */
        box-sizing: border-box;
      }

测试三列布局, 不要使用 inline-block
